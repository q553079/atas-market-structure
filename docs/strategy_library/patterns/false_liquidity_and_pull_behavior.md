# False Liquidity And Pull Behavior

## Status

- `status`: `doctrine_accepted`
- `source_type`: `displayed-liquidity behavior doctrine synthesis`

## Pattern Summary

This pattern is for situations where displayed size appears important, attracts attention, and then either disappears, relocates, or fails to produce the expected defense.

The important distinction is:

- displayed size is not yet support or resistance
- only lifecycle plus outcome can promote it into real structure

This card is aimed at fake opportunity control.
It helps the system say:

- this looked important
- but the behavior was mostly visual bait

## Observed Facts To Preserve

- when the displayed order first appeared
- whether it stayed, moved, shrank, or pulled as price approached
- whether aggressive volume actually traded into it
- whether price reaction happened after interaction
- whether the same behavior repeated at the same price or only flashed once

## Derived Interpretation

These belong in `derived interpretation`:

- `false_liquidity_attention_trap`
- `pull_before_touch`
- `displayed_size_without_defense`
- `visual_pressure_not_real_control`
- `fake_support_or_resistance_candidate`

## Confirmation Signals

- the displayed order pulls or relocates before meaningful interaction
- price does not react the way a truly defended level should react
- repeated flashes of size fail to create persistent support or resistance behavior

## No-Trade Conditions

- do not lean on displayed size alone without lifecycle evidence
- do not call a level defended if the size repeatedly pulls before pressure arrives
- do not fade price only because a large wall appears if that wall has not produced outcome before

## Management Notes

- this card is best used with live depth and trapped-order review
- a false-liquidity read gets stronger when repeated visual size keeps failing to produce real defense
- if size starts staying and producing outcome, hand the read off to a real defense or control card

## Review Questions

- what actually happened to the visible size when price got close
- did the operator trade the size itself or the outcome it produced
- was the supposed wall or bid only visual bait that disappeared under pressure
